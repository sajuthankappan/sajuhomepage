Ubuntu New User Tips

Open Run Application Dialog

Press Alt + F2

Open Terminal (Command Prompt)

Applications -> Accessories -> Terminal
  or
Press Alt + F2 and type gnome-terminal

Open Task Manager

GUI

System -> Administration -> System Manager

Installing Flash Plugin

GUI:

Run Synaptic and install "flashplugin-installer"

CLI:

sudo apt-get install flashplugin-installer

Installing MP3 Codec

Use File Manager and double click on any mp3 file and follow system prompts to install codecs

Watching (Encrypted) DVDs

CLI:

sudo apt-get install libdvdcss2
sudo /usr/share/doc/libdvdread4/install-css.sh

Alternate Way: (Using Medibuntu Repository)

GUI:

Run Synaptic and add the following medibuntu repository (Settings -> Repository -> Other Softwre -> Add)
deb http://packages.medibuntu.org/ jaunty free non-free
Press Reload toolbar button

Install the following packages
libdvdcss2 libdvdread4

CLI:

sudo gedit /etc/apt/sources.list
and add the following line at the end:
deb http://packages.medibuntu.org/ jaunty free non-free
sudo apt-get update
sudo apt-get install libdvdcss2 libdvdread

Gaming in Ubuntu

Ubuntu is not suited for hardcore gamers. Some windows games can be made to run using the application 'wine' (A windows api emulator). But the results may vary.
See http://appdb.winehq.org and http://www.winehq.org for more details
